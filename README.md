# Not Today Machariel!

A browser tool for finding **anti-smartbomb bookmarks** in EVE Online.

Gate camps work because a bookmark on the straight line between two gates makes
your warp-in completely predictable — you land where a camp is already waiting.
This tool finds bookmark positions set at an angle *off* those gate-to-gate lines,
so you don't drop straight into the fireworks. Pick a system, set how much
clearance you want, and it gives you the exact in-game warp steps to build the
bookmark.

It's free and runs entirely in your browser — no account, no download, no server.

## What it is (and isn't)

It's pure geometry: a planning aid, not a guarantee. It can't see cloaks, bubbles,
or your d-scan, and single-gate systems are flagged as "not evaluated" rather than
pretended safe. Use it to plan smarter bookmarks — your own judgement still matters.
